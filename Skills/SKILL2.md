# Research Report skill

When asked to research a topic, produce a tight, structured brief rather than a wall
of text:

1. **Bottom line first** — 2–3 sentences answering the question directly.
2. **Key findings** — the handful of points that actually matter, each one line.
3. **Detail** — only where it earns its place; cite sources by name.
4. **What's uncertain / what to check next** — be honest about gaps.

Prefer primary sources. Flag anything that looks like marketing or is unverified.
If results conflict, say so rather than smoothing it over. Save the finished brief
to the markdown vault so it becomes future memory.
